ASICE Viewer

Használati útmutató

Adatkezelés

  • Jelen oldal nem tárol vagy gyűjt semmilyen adatot és nem továbbít adatokat harmadik félnek. A kiválasztott fájlok a gép memóriájába töltődnek be és onnan kerülnek feldolgozásra offline módon.

Használat

  1. Először válasszon nyelvet, hogy milyen nyelvű felületet kíván használni
  2. A megjelenő oldalon válassza ki a megnyitandó fájlt!
  3. A megjelenő fájl listából válassza ki a megtekintendő PDF fájlt és az megjeleni az oldalon
   Usage information

Data handling

  • This page does not collects or transfers any data about you. The selected file is only loaded into your computer's memory and processed within the browser offline.

Usage

  1. Select the language you would like to use for the UI
  2. Select the file to be viewed on the next page
  3. From the displayed list select that PDF file you want to view and it will be displayed on the page
